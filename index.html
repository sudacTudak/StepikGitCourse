git checkout (co) . Удалит изменения, еще не залитые в стейдж

git reset . Вернет все файлы из stage. Для удаления изменений вернись на шаг 1

git reset --hard HEAD^1 - удалит изменения последнего коммита, передвинет указатель на коммит, стоящий перед удаленным

git reset --soft HEAD^1 - вернет изменеия последнего коммита в stage, передвинет указатель на коммит, стоящий перед удаленным. Чтобы вытащить из stage - вернись на шаг 2
